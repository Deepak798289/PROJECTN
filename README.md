# Projectn

### set up instruction
1.clone project
```
   git remote add origin https://github.com/Deepak798289/PROJECTN.git
```
2.move into directory
```
 cd PROJECTN
```
3. install dependencies
```
   npm i

```
4.run the server
```
 npm run dev
```
5.install tailwindcss
```
  .npm install -D tailwindcss
  .npx tailwindcss init
```
